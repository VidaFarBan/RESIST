About the project

The RESIST visualisation tool shows a set of solution, tools and technologies which has been used towards the climate change adaptation in different regions in Europe. It was developed as part of the RESIST project. Information on Climate Change Adaptation Solutions, Tools and Technologies , performed under the RESIST Part of the RESIST study

This tool is best viewed using Google Chrome or Firefox.

Warning!

Only one case can be pinned at a time. If you pin this one, the previous pinned item will be discarded.

Select a case from the list or the map to be able to zoom into it.
